Interview Review Topics

What is a monorepo setup, and why do companies use it?
A repository holding code for multiple projects (e.g. client and server). It makes package sharing, atomic changes, and unified builds simple.
Why choose Vite over Create React App (CRA)?
Vite pre-bundles dependencies with esbuild (Go-based) and serves source code over native ESM, loading dev servers instantly.
